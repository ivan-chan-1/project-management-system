
Course Creation
Course administrators can create courses on the platform and invite students via e-mail.
![Subcourse Creation](https://github.com/user-attachments/assets/691c983d-25f5-4b12-94a3-12c52c0d796e)

Clients and Project Creation
Clients can find and join published courses where they want students to help develop a project. Once approved, clients can create project specifications.
![Client](https://github.com/user-attachments/assets/21929421-51dc-4a7d-938f-ef91b9993922)

Students and Group Creation
Students can use the platform to find a group to work with. Students can preference projects they would like to work on. When deciding on the group's project preference, group members can auto-rank based on their personal preferences. 
![Finding Group and Project Preferences (1)](https://github.com/user-attachments/assets/c1bf03a0-19c6-44ce-93df-03960ea2d9f3)

Allocating Groups to Projects
Course administrators can allocate groups to projects based on a selected allocation algorithm or manually allocate projects.
![Uploading Groups to Projects.gif…]()

Communication
When allocations of projects have been finalised, students, clients and course staff can use the platform to communicate.
